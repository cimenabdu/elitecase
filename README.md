# ELITE CASE

Site e-commerce premium noir et blanc pour une marque de coques de téléphone personnalisées.

## Importer sur GitHub

1. Téléchargez le ZIP fourni.
2. Décompressez-le.
3. Créez un dépôt GitHub nommé `elite-case`.
4. Cliquez sur **Add file > Upload files**.
5. Glissez tous les fichiers et dossiers du projet dans GitHub.
6. Committez les fichiers.
7. Allez dans **Settings > Pages**.
8. Dans **Build and deployment**, choisissez **GitHub Actions**.
9. Le site sera publié automatiquement après le workflow.

## Lancer en local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
